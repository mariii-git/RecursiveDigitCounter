Recursive Digit Counter (C#)

A minimal, recursive C# function that counts the digits of an integer.

Features

· Pure recursive implementation
· Handles any positive integer
· Clean and readable logic
· No external dependencies

Technologies

· C# (.NET)

Usage

csharp
// Call the method
Console.WriteLine(DigitCount(1234)); // Output: 4
Console.WriteLine(DigitCount(7));    // Output: 1
Console.WriteLine(DigitCount(987654321)); // Output: 9
